📄 About this Toolkit

AlgoLens is a frontend interface for three standalone Python algorithm analysis tools. Each tool is implemented in pure Python and exposed here as an interactive UI.

  • asymptotic_analysis.py — empirical complexity profiling via regression fitting
  • coordinate_transformation.py — homogeneous matrix-based 2D transformations
  • recursive_check.py — static AST-based recursion and complexity estimation

⚡ Quick Start

Select a tool from the sidebar. Each tool has an input area and a results panel. For the Asymptotic Analyzer, paste any Python function or snippet and click Run Analysis. For Coordinate Transform, enter a point and choose a transformation type. For Recursion Analyzer, paste any Python code with functions.
